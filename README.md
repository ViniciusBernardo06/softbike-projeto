# 🚲 SOFTBIKE — Projeto de Desenvolvimento Web

Site institucional fictício de uma loja de bicicletas, desenvolvido como projeto acadêmico para a disciplina de **Desenvolvimento Web** do curso de **Análise e Desenvolvimento de Sistemas (ADS)**.

---

## 📋 Sobre o Projeto

O **Softbike** é um site multi-página criado com HTML, CSS e Bootstrap 5, simulando o portal de uma loja de bicicletas. O projeto foi estruturado para demonstrar na prática os principais conceitos de desenvolvimento web front-end, como responsividade, navegação entre páginas, exibição de produtos e tabelas de dados.

---

## 📁 Estrutura de Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Página inicial (Home) |
| `empresa.html` | Informações sobre a empresa |
| `produtos.html` | Catálogo de produtos |
| `texto.html` | Página de textos/conteúdo editorial |
| `listas.html` | Demonstração de listas HTML |
| `tabela.html` | Tabela de dados/preços |
| `links.html` | Página de links úteis |
| `wireframe.html` | Documentos e wireframes do projeto |
| `style.css` | Estilos personalizados do projeto |

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização customizada via `style.css`
- **Bootstrap 5.1.3** — framework CSS para layout responsivo e componentes

---

## 🚀 Como Executar

Não é necessário instalar nada. Basta clonar o repositório e abrir os arquivos no navegador:

```bash
git clone https://github.com/ViniciusBernardo06/softbike-projeto.git
cd softbike-projeto/softbike_bootstrap
```

Em seguida, abra o arquivo `index.html` diretamente no seu navegador, ou use a extensão **Live Server** no VS Code para uma melhor experiência de desenvolvimento.

---

## 📌 Observações

- O projeto foi reorganizado para seguir o modelo Bootstrap apresentado em sala de aula.
